This project, RumourSense AI, is a Neuro-Fuzzy based early rumour spread prediction system designed to identify whether a given piece of text from social media is a rumour or genuine information. The system combines machine learning techniques with fuzzy logic to provide both classification and interpretability. It uses text preprocessing and TF-IDF vectorization along with a Logistic Regression model to predict the probability of a rumour. Based on this probability, a fuzzy logic layer categorizes the potential spread risk into low, medium, or high levels. The application is implemented using Python and deployed through a Streamlit-based web interface, allowing users to input text and instantly view results such as final verdict, risk index, and threat level. This project demonstrates the integration of artificial intelligence and fuzzy systems to address the real-world problem of misinformation on social media platforms.
